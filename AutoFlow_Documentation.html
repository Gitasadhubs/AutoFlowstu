üöÄ Deployment Guide

üéØ Production Deployment Options

Choose from multiple production-ready deployment strategies based on your requirements.

Option 1: Vercel + Railway (Recommended)

Frontend: Vercel

Steps:

  1. Connect GitHub repository to Vercel
  2. Configure build settings: npm run build
  3. Set environment variables for API URL

Backend: Railway

Steps:

  1. Deploy API server to Railway
  2. Configure PostgreSQL database
  3. Set all required environment variables

Required Environment Variables

# Database Configuration
DATABASE_URL=postgresql://user:password@host:5432/database

# GitHub OAuth Configuration
GITHUB_CLIENT_ID=your_github_oauth_client_id
GITHUB_CLIENT_SECRET=your_github_oauth_client_secret

# Session Configuration
SESSION_SECRET=your_cryptographically_secure_random_string

# Application Configuration
NODE_ENV=production
PORT=5000
